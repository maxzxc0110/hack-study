# 外部侦查

这章主要讲信息收集

侦查分主动侦查和被动侦查

## 主动侦查
各种扫描

## 被动侦查
借助 Google、LinkedIn、Shodan 和社交媒体进行信息收集


# DNS

dig命令
```
dig cyberbotic.io +short
```

whois命令
```
whois 104.21.90.222
```

[Spoofcheck](https://github.com/BishopFox/spoofcheck)
```
./spoofcheck.py cyberbotic.io
```

# 社交媒体
谷歌黑语法，通过领英收集信息
```
 site:"linkedin.com" "<company name>"
```
