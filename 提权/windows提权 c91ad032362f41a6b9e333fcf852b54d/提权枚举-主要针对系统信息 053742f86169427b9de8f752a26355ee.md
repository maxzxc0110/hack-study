# 提权枚举-主要针对系统信息

**windows版本和配置：**

```bash
systeminfo | findstr /B /C:"OS Name" /C:"OS Version"
```

查看补丁：

```bash
wmic qfe
```

用户枚举：

```bash
whoami /privwhoami /groups
```

查看所有用户：

```bash
net userwhoami /allGet-LocalUser | ft Name,Enabled,LastLogonGet-ChildItem C:\Users -Force | select Name
```
